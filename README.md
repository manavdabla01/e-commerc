# 🌐 Modernshop - E-commerce Website UI

Modernshop is a responsive and stylish e-commerce frontend project built using **HTML**, **CSS**, and **JavaScript**. It simulates an online shopping interface with filtering, searching, and cart simulation.

---

## 🚀 Features

- 🛍️ Product listing with images, prices, and names  
- 🔍 Product search and category filtering  
- 🛒 Simulated cart functionality with JavaScript  
- 📱 Mobile-friendly responsive design  
- ⚡ Fast and lightweight

---

## 🛠️ Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript**
